Flow: Modularized Agentic Workflow Automation (ICLR'25)

A multi-agent framework designed for efficient task automation through modular workflow design and dynamic refinement.

Key Features

Modular Workflow Design

Workflows are structured as Activity-on-Vertex (AOV) graphs, enabling parallel execution while minimizing dependencies.

Concurrency

Subtasks execute concurrently based on predefined dependencies.

Dynamic Workflow Refinement

The framework adapts workflows in real-time based on performance feedback.

Error Handling & Recovery

Auto-validation of subtasks ensures workflow resilience, preventing failures during execution.

How It Works

  • Workflow Initialization: Tasks are structured into an AOV graph, and the system selects the most efficient structure.
  • Execution & Dynamic Updates: Agents process subtasks concurrently and update workflows dynamically during execution.
  • Workflow Refinement: Failed tasks are reassigned or redesigned to prevent bottlenecks.
  • Subtask Validation: Validation is performed based on subtask type and feedback, and results are used to improve subsequent executions.

Installation & Usage

Requirements

  • Python 3.8+
  • OpenAI API

Installation

Clone the repository and install the necessary dependencies:

git clone https://github.com/tmllab/2025_ICLR_FLOW.git
cd 2025_ICLR_FLOW
pip install -r requirements.txt

Configuration

Set your API key and model for configuration:

export OPENAI_API_KEY="your-api-key"

Set OpenAI Model and define your task:

GPT_MODEL: str = "your-model"
overall_task: str = "your-task"

Run Flow

python main.py
Visit the GitHub Repository
